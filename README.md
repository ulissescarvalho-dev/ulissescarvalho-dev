## 📌 Sobre mim

* 🚀 **Foco Atual:** Mergulhando fundo no ecossistema **Java** e nos pilares da Programação Orientada a Objetos (POO).
* 📦 **Padronização:** Utilizo o **Maven** para gerenciar ciclos de vida e dependências, garantindo que meus projetos sigam padrões profissionais de mercado desde o início.
* 🐧 **Ambiente Unix & Virtualização:** Desenvolvo nativamente em ambiente **Unix**, utilizando **Máquinas Virtuais** como laboratórios isolados para explorar diferentes distribuições Linux, configurar servidores e realizar experimentos de infraestrutura com segurança.
* 🧠 **Estudo Autônomo:** Autodidata por natureza. Transformo cada novo conceito em um repositório prático, documentando toda a minha evolução técnica aqui no GitHub.
