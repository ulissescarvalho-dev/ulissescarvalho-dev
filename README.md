### 🐧 Sobre o meu ecossistema de desenvolvimento

Diferente de um fluxo de trabalho comum em Windows, eu utilizo o **WSL2 (Windows Subsystem for Linux)** com **Ubuntu**. Isso me permite:
* Desenvolver nativamente em ambiente **Unix**.
* Utilizar o terminal Linux para automação e laboratórios de infraestrutura.
* Garantir performance e compatibilidade máxima com ferramentas de Backend.
